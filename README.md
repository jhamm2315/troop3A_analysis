# Troop 3A Tactical Traffic Analysis

**Author:** Justynn Hammond  
**Role:** Senior Data Analyst

## Overview
This repository contains my tactical traffic analysis of Troop 3A Q2 crash activity across 2021-2025. The work focuses on corridor-level crash patterns, time-based risk, causal-factor analysis, and hotspot identification to support deployment decisions.

## Contents
- `notebook/` reproducible Jupyter notebook
- `report/` executive-style report outputs
- `charts/` visuals used in the report
- `data/` placeholder only; raw source data is not redistributed

The original source spreadsheet is not included in this repository because it was provided as part of a candidate exercise and is not redistributed here.
